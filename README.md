# Social-Media-Engagement-Analytics-Python
Python project analyzing social media engagement data — cleaning, wrangling, visualization, and insights on user behavior and content performance

# 📱 Social Media Engagement Analytics (Python Project)

**Tools:** Python | Pandas | NumPy | Matplotlib | Seaborn | Plotly  
**Domain:** Social Media | Data Analytics | Visualization | User Behavior  

---

## 📊 Project Overview
This project analyzes a dataset of **5,000 social media engagement records** to uncover insights into:
- Content performance by post type, category, and country
- User trends by age, verification status, and sentiment
- Behavioral patterns such as best posting time and device impact
- Sentiment analysis and its effect on engagement

The analysis was performed using **Python for Data Analysis**, leveraging data cleaning, wrangling, statistical analysis, and visualization.

---

## 🎯 Problem Statement
Social media platforms generate massive engagement data (likes, comments, shares, impressions, watch time).  
The goal was to:
1. Clean and prepare raw engagement data.  
2. Explore and summarize metrics using Pandas.  
3. Perform statistical analysis on engagement KPIs.  
4. Visualize trends with Matplotlib, Seaborn, and Plotly.  
5. Generate actionable insights for content strategy.  

---

## 📂 Dataset
- **File:** `social_media_engagement_5000.csv`  
- **Attributes:**  
  - post_id, post_type, category, country  
  - likes, comments, shares, impressions, watch_time  
  - engagement_rate, followers, sentiment, device, posted_date  

---

## 🔧 Python Techniques Used
- **Data Import & Setup:** `read_csv`, dtype checks, datetime conversion  
- **Data Cleaning:** `fillna()`, `dropna()`, duplicates removal, category standardization  
- **Feature Engineering:** hashtag count, engagement_score, log‑transformed metrics  
- **Exploration:** `describe()`, `value_counts()`, correlation matrix, groupby summaries  
- **Statistical Analysis:** mean, median, mode, variance, percentiles  
- **Visualization:**  
  - *Matplotlib*: scatter, line, bar, pie, histogram, box  
  - *Seaborn*: count plot, bar plot, violin, pair plot, heatmap, swarm plot  
  - *Plotly*: interactive line, bar, bubble, scatter charts  

---

## 📊 Key Insights
1. **Content Performance**  
   - 🎥 Video posts achieve the highest engagement rates.  
   - 📊 Analytics & Programming categories lead in engagement.  
   - 🌍 Brazil, Australia, and France show strongest regional traction.  

2. **User Trends**  
   - 👩‍🎓 Engagement peaks among younger users (18–25).  
   - ✅ Verified accounts drive significantly higher engagement.  

3. **Behavioral Insights**  
   - 🕕 Best posting time: **6 PM – 9 PM** (peak impressions).  
   - 📱 Mobile dominates watch time compared to desktop.  

4. **Sentiment Analysis**  
   - 😀 Positive sentiment posts yield strongest engagement.  
   - 😐 Neutral posts perform moderately.  
   - 😡 Negative sentiment posts underperform across all metrics.  

---

## 📌 Conclusion
This project demonstrates how **Python for Data Analysis** can transform raw social media data into actionable insights.  
Findings highlight the importance of **video content, youth‑focused strategies, verified accounts, mobile optimization, and positive sentiment** in driving engagement.
